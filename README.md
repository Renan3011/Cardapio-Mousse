# 🍫 Cardápio Digital - Propósito & Sabor

Projeto de um cardápio digital desenvolvido com HTML e CSS, com foco em apresentar produtos de forma organizada, moderna e visualmente atrativa.

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  

---

## 💻 Sobre o projeto

Este projeto foi desenvolvido como parte do meu aprendizado em desenvolvimento web.

A proposta foi criar uma interface simples e funcional para exibição de produtos (mousses), incluindo:

- Banner principal
- Cards de produtos
- Imagens ilustrativas
- Botões de ação
- Estilização com foco em experiência do usuário

---

## 📈 Próximos passos

- Implementar JavaScript  
- Adicionar interatividade (ex: carrinho de compras)  
- Melhorar responsividade para dispositivos móveis
